# vk-ym-playlist-transfer
Скрипты для переноса плейлистов из ВК в ЯМ(яндекс.музыка). 

Все шаги - через браузер (Фаерфокс/Хром)
Чтобы остановить работу скрипта - обновите страницу (сбросится и скрипт и переменные)

Шаг 1: экспорт списка композиций из ВК. 
* Открываете свой плейлист в ВК
* Открываете консоль (F12, Ctrl+Shift+i)
* В поле консоли вставляете скрипт1 и отправляете(Enter)
* Скрипт проскроллит ваш плейлист, чтобы подгрузились все треки
* Достав названия треков - сохранит файлом в браузере (vk-playlist.txt)

Шаг 2: импорт поштучно композиций в указанный плейлист в ЯМ.
* Открываете свою ЯМ на главной странице, где поле поиска видно
* (опционально) создаёте новый плейлист `"New playlist"`
* Открываете консоль (F12, Ctrl+Shift+i)
* В поле консоли вставляете:
    window.__YM_PLAYLIST_NAME = "New playlist";
  Отправили
    window.__YM_TRACKS = `[сюда вставляете содержимое файла, полученного на шаге1]`;
  Отправили
* В поле консоли вставляете скрипт2 и отправляете(Enter)
* Скрипт будет каждую композицию искать, и если яндекс находит такой трек - добавлять ему лайк и в плейлист, который вы указали.
* Все треки, которые яндекс не нашёл у себя - снова загрузятся в виде файла (ym-not-found.txt)

Шаг 3: проверка результатов
* Треки уже в плейлисте. Можно повторить шаги 1 и 2 для остальных плейлистов.
* Ненайденные треки можете попробовать поискать самостоятельно в ЯМ, особенно если в названиях треков были лишние символы, ошибки или приписки, которые сбивают поиск
* Иногда в результате поиска яндекс возвращяет иной трек, чем ищется. Он будет добавлен. Такое можно увидеть только глазами, - в скрипте такая ситуация не решается.
